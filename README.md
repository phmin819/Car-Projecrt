# 🚗 3D Car Parking Game

안드로이드 웹뷰(WebView) 환경에서 구동되도록 개발된 **웹 기반 3D 주차 시뮬레이션 게임**입니다. 
가벼우면서도 현실감 넘치는 물리 엔진과 다양한 난이도/차량 선택 시스템을 제공합니다.

## ✨ 주요 기능 (Features)

* **다양한 차량 선택 시스템:** 
  * 🚙 `COMPACT`: 작고 회전이 빨라 초보자에게 적합
  * 🚓 `SEDAN`: 밸런스가 잡힌 표준 차량
  * 🚚 `TRUCK`: 크기가 크고 조작이 어려워 고수용
* **난이도 해금(Unlock) 시스템 & 테마 맵:**
  * 이전 난이도를 클리어해야 다음 난이도가 해금되는 프로그레션 시스템 적용 (`localStorage` 활용).
  * 🌲 `EASY`: 평화로운 숲 테마
  * 🌃 `MEDIUM`: 야간 네온 시티 테마 (EASY 클리어 시 해금)
  * 🌋 `DIFFICULT`: 극한의 용암 지대 테마 (MEDIUM 클리어 시 해금)
* **사실적인 차량 물리 (Physics):**
  * Ackermann Steering(애커먼 조향) 방식을 근사하여 현실적인 차량 회전 반경 구현
  * 가속, 감속, 최고 속도 및 핸들 복원력 적용
* **3D 렌더링 & UI:**
  * `Three.js`를 활용한 3D 차량 모델링, 동적 그림자(Shadow), 환경 파티클 효과 렌더링
  * 실시간 차량의 위치와 각도를 보여주는 2D 미니맵(레이더) 지원
  * 직관적인 페달 조작 및 스와이프 방식의 핸들 컨트롤러

## 🛠 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **3D Graphics:** [Three.js](https://threejs.org/)
* **Platform:** Android Studio (WebView 기반 패키징)

## 🎮 게임 방법 (How to Play)

1. 스플래시 화면이 지나가면, 플레이할 **차량**을 선택합니다.
2. 플레이할 **난이도**를 선택합니다. (처음에는 EASY만 선택 가능)
3. 좌측 하단의 스티어링 휠(핸들)을 터치하여 드래그하면 바퀴가 꺾입니다.
4. 우측 하단의 전진(FORWARD) / 후진(BACKWARD) 페달을 눌러 이동합니다.
5. 화면의 초록색 **주차 구역**에 차량을 반듯하게 주차하면 클리어됩니다.
6. 제한된 시간이 초과하거나 장애물에 부딪히면 게임 오버됩니다.
